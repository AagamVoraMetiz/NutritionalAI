# **App Name**: NutriPlan

## Core Features:

- User Profile Input: Collect user data including age, gender, height, weight and BMR.
- Calorie Calculation: Calculate daily caloric needs based on user data and weight goals (loss, maintain, or gain).
- Meal Plan Generation: Generate a sample daily meal plan based on the user's caloric needs and preferences, using a tool to pick from a variety of food items.
- Meal Plan Display: Display the generated meal plan with nutritional information for each meal.
- Serving Size Adjustment: Allow users to manually adjust serving sizes in the meal plan to customize caloric and macro-nutrient intake.

## Style Guidelines:

- Primary color: Green (#4CAF50) to represent health and nutrition.
- Secondary color: Light Gray (#F0F0F0) for backgrounds and neutral elements.
- Accent: Orange (#FF9800) for call-to-action buttons and highlights.
- Clean and readable typography with clear hierarchy.
- Use simple and intuitive icons to represent different food groups and actions.
- Clean and well-spaced layout for easy readability and navigation.

## Original User Request:
An app that creates nutritional plans for user where it collects data like age, gender, height, weight, BMR, weight loss or gain (Optional) and stores the data in firebase. Create using React-native technology.
